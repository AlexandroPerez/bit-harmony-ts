## React Component for rendering text

A basic card container that renders the provided text, alongside a customizable
button.

### Component usage

```js
<Card text="hello from Card" importance="primary" />
```

### Using props to customize the text

Modify the text to see it change live:

```js live
<Card text="hello from Card" importance="primary" />
```
